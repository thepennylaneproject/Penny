## Title

Orchestration enqueue secret can be read from sessionStorage by page scripts

## Description

apiFetchWithEnqueueSecret merges a user-supplied enqueue secret from sessionStorage into Authorization headers. Any XSS in the dashboard origin can read sessionStorage and exfiltrate the secret.

## Proof hooks

- **[code_ref]** Session storage value is attached as Bearer token when present.
