## Title

GitHub webhook accepts any signature header without HMAC verification

## Description

The Supabase Edge function for GitHub webhooks rejects requests in production only when the x-hub-signature-256 header is missing. It never computes HMAC-SHA256 over the raw body with GITHUB_WEBHOOK_SECRET or compares it to the header.

## Proof hooks

- **[code_ref]** Production only requires presence of signature header; no HMAC verify implementation before processing body.
