## Title

Supabase helpers use select('*') with no row limits

## Description

Dashboard helpers fetch entire rows for projects, audit_runs, findings, model_usage, webhooks, and schedules. Large JSON columns (metadata, proof_hooks, history) inflate payloads and memory on the server and client. getFindings has no default LIMIT, so projects with many findings pay full-table transfer per call.
